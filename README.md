# CYOA Horror Project

This is an interactive "Choose Your Own Adventure" horror story inspired by *Don't Breathe*.

## Description
This project is a browser-based horror experience where the player makes choices that affect the outcome of the story. Each decision leads to different paths and multiple endings such as escape, survival, or capture.

The project focuses on suspense, tension, and limited information to create an immersive experience similar to horror films.

## How to Play
1. Open the website link
2. Read the story carefully
3. Click the buttons to make decisions
4. Each choice leads to a different outcome

## Features
- Multiple story paths
- Different endings
- Interactive decision making
- Horror themed design

- HTML
- CSS
